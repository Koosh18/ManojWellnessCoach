# Herbal Life Coach - Manoj Gupta

Modern, premium, conversion-focused single-page website for **Herbal Life Coach Manoj Gupta**, focused on natural weight loss, herbal nutrition, and lifestyle transformation.

## Tech Stack

- **HTML5**
- **Tailwind CSS** via CDN (no build step required)
- **Vanilla JavaScript** for animations, testimonial carousel, and form validation

## How to Run Locally

1. Make sure you are in the project folder:

   ```bash
   cd "C:\Users\Koosh Gupta\herbal_life_coach"
   ```

2. Start a simple HTTP server (choose one option):

   - Using **Python 3** (if installed):

     ```bash
     python -m http.server 8000
     ```

   - Or using **Node.js** with `npx` (if installed):

     ```bash
     npx serve . --listen 8000
     ```

3. Open the site in your browser:

   ```text
   http://localhost:8000
   ```

## Customization

- Edit `index.html` to change copy, colors, or layout.
- Update WhatsApp links, phone number, or email directly in the HTML if needed.
